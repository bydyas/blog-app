## Backend setup:

! Copy **.env.example** to **.env.development**

```bash
# install packages
$ npm install

# compose db container (docker)
$ npm run start:db

# run app
$ npm run start:dev
```

## Frontend setup:

! Copy **.env.example** to **.env**

```bash
# install packages
$ npm install

# run app
$ npm run dev
```
